SEO Optimization Plan for Patro Seguros

Phase 1: Advanced Structured Data (JSON-LD)
- Implement dynamic 'Service' and 'Product' schemas for all landing pages.
- Integrate 'AggregateRating' directly into product schemas to show stars in Google search.
- Ensure 'LocalBusiness' schema is consistent across all local pages.

Phase 2: Hyper-Local Internal Linking
- Update Footer to include a 'Neighborhood Hub' (Bairros de Guarulhos) to capture long-tail local traffic.
- Improve 'SmartText' utility to automatically link service names to their respective landing pages.
- Optimize the 404 page to reduce bounce rate by suggesting popular insurance products.

Phase 3: Technical SEO & Metadata
- Audit and add missing 'alt' tags to core images (Selo Melhor Corretora, Logos).
- Normalize canonical URLs in 'PageMeta' to prevent duplicate content issues.
- Refine 'robots.txt' to optimize crawl budget and protect sensitive routes.

Phase 4: Content Enrichment
- Expand landing page data in 'landingPages.ts' with unique meta descriptions and contextual FAQs.
- Implement 'Breadcrumb' schema for all service paths.

Technical Details:
- Components: PageMeta, InsurancePageTemplate, Footer, SmartText.
- Data: landingPages.ts.
- Assets: robots.txt, sitemap.xml updates.
